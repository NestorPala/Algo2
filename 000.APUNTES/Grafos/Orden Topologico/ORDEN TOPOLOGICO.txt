ORDEN TOPOLOGICO

* Se utiliza para resolver problemas de precedencia (unas tareas antes que otras)
* No puede haber ciclos (dependencia circular de las tareas)
* El grafo tiene que ser dirigido
* Se resuelve usando el algoritmo de orden de entrada (similar BFS) o similar DFS

* El algoritmo de orden de entrada se puede resolver usando Cola o Pila; con pila el orden queda menos paralelizable y mas lineal, y con cola el orden queda mas paralelizable y menos lineal; depende del caso nos conviene una implementacion u otra

* Para detectar ciclos nos conviene utilizar grados de entrada; con DFS tambien se puede pero es mas dificil
